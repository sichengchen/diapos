# Section

Marker component for grouping slides. Deck recognizes Section children during tree flattening and auto-generates divider slides.

**Source:** `packages/diapos/src/core/Section.tsx`

## Import

```ts
import { Section } from 'diapos'
import type { SectionProps } from 'diapos'
```

## Props

```ts
interface SectionProps {
  title: string
  subtitle?: string
  children: ReactNode
}
```

## Behavior

Section is a marker — it renders `<>{children}</>` (passthrough).

When `parseSlides()` encounters a Section element during Deck initialization:

1. Auto-generates a divider `<Slide>` using the section's `title` and `subtitle`
2. Includes the section's children as subsequent slides in the flat array

## Valid Usage

```tsx
<Deck>
  <Section title="Part 1" subtitle="Introduction">
    <Slide>...</Slide>
    <Slide>...</Slide>
  </Section>
  <Section title="Part 2">
    <Slide>...</Slide>
  </Section>
</Deck>
```

## Constraints

- Must be a direct child of `<Deck>`.
- Children are `<Slide>` components.
- `title` is required.
- Sections do not nest.
- The divider slide is generated by `parseSlides`, not by Section itself.
