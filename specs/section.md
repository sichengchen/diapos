# Section

## Import

Not yet exported from the public API. Available internally at:

```ts
import { Section } from './components/Section'
```

## Props

```ts
interface SectionProps {
  title: string
  subtitle?: string
  children: ReactNode
}
```

## Behavior

- Groups slides logically. `<Deck>` recognizes `<Section>` children during tree flattening.
- When `<Deck>` encounters a `<Section>`, it auto-generates a divider `<Slide>` using the section's `title` and `subtitle`, then includes the section's children as subsequent slides.
- Renders `<>{children}</>` as a fallback (passes children through).

## Valid Usage

```tsx
<Deck>
  <Section title="Part 1" subtitle="Introduction">
    <Slide>...</Slide>
    <Slide>...</Slide>
  </Section>
  <Section title="Part 2">
    <Slide>...</Slide>
  </Section>
</Deck>
```

- Must be a direct child of `<Deck>`.
- Children should be `<Slide>` components.
- `title` is required.

## Constraints

- `<Section>` does not render its own frame — `<Deck>` handles divider generation.
- Sections do not nest (no `<Section>` inside `<Section>`).
- The divider slide is auto-generated by `<Deck>`, not by `<Section>` itself.
